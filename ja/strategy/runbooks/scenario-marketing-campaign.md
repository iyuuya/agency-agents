# 📢 ランブック: マルチチャネルマーケティングキャンペーン

> **モード**: NEXUS-マイクロ 〜 NEXUS-スプリント | **期間**: 2〜4週間 | **エージェント数**: 10〜15

---

## シナリオ

複数のチャネルにまたがる統合マーケティングキャンペーンを展開します。コンテンツはプラットフォームごとに最適化され、ブランドの一貫性を保ち、データに基づいたものでなければなりません。キャンペーンは測定可能な獲得とエンゲージメントを促進する必要があります。

## エージェントロスター

### キャンペーンコア
| エージェント | 役割 |
|-------|------|
| ソーシャルメディアストラテジスト | キャンペーンリード、クロスプラットフォーム戦略 |
| コンテンツクリエーター | すべてのフォーマットにわたるコンテンツ制作 |
| グロースハッカー | 獲得戦略、ファネル最適化 |
| ブランドガーディアン | すべてのチャネルにわたるブランド一貫性 |
| アナリティクスレポーター | パフォーマンストラッキングと最適化 |

### プラットフォームスペシャリスト
| エージェント | 役割 |
|-------|------|
| Twitter エンゲージャー | Twitter/X キャンペーンの実行 |
| TikTok ストラテジスト | TikTok コンテンツと成長 |
| Instagram キュレーター | Instagram ビジュアルコンテンツ |
| Reddit コミュニティビルダー | Reddit における本物のエンゲージメント |
| アプリストアオプティマイザー | アプリストアのプレゼンス（モバイルの場合） |

### サポート
| エージェント | 役割 |
|-------|------|
| トレンドリサーチャー | 市場タイミングとトレンドの整合 |
| エクスペリメントトラッカー | キャンペーンバリエーションの A/B テスト |
| エグゼクティブサマリージェネレーター | キャンペーンレポーティング |
| 法的コンプライアンスチェッカー | 広告コンプライアンス、開示要件 |

## 実行計画

### 1週目: 戦略とコンテンツ制作

```
1〜2日目: キャンペーン戦略
├── Social Media Strategist → クロスプラットフォームキャンペーン戦略
│   ├── キャンペーン目標と KPI
│   ├── ターゲットオーディエンスの定義
│   ├── プラットフォーム選定と予算配分
│   ├── コンテンツカレンダー（4週間計画）
│   └── プラットフォームごとのエンゲージメント戦略
│
├── Trend Researcher → 市場タイミング分析
│   ├── 合わせるべきトレンドトピック
│   ├── 競合キャンペーン分析
│   └── 最適なローンチタイミング
│
├── Growth Hacker → 獲得ファネルの設計
│   ├── ランディングページ最適化計画
│   ├── コンバージョンファネルのマッピング
│   ├── バイラルメカニズム（リファラル、シェア）
│   └── チャネル予算配分
│
├── Brand Guardian → キャンペーンブランドガイドライン
│   ├── キャンペーン固有のビジュアルガイドライン
│   ├── メッセージングフレームワーク
│   ├── キャンペーンのトーンとボイス
│   └── すべきこととすべきでないこと
│
└── Legal Compliance Checker → 広告コンプライアンスレビュー
    ├── 開示要件
    ├── プラットフォーム固有の広告ポリシー
    └── 規制上の制約

3〜5日目: コンテンツ制作
├── Content Creator → マルチフォーマットコンテンツ制作
│   ├── ブログ投稿 / 記事
│   ├── メールシーケンス
│   ├── ランディングページのコピー
│   ├── 動画スクリプト
│   └── ソーシャルメディアコピー（プラットフォーム別に最適化）
│
├── Twitter Engager → Twitter 固有のコンテンツ
│   ├── ローンチスレッド（10〜15ツイート）
│   ├── 日次エンゲージメントツイート
│   ├── 返信テンプレート
│   └── ハッシュタグ戦略
│
├── TikTok Strategist → TikTok コンテンツ計画
│   ├── 動画コンセプト（3〜5本）
│   ├── フック戦略
│   ├── トレンドの音楽/フォーマットとの整合
│   └── 投稿スケジュール
│
├── Instagram Curator → Instagram コンテンツ
│   ├── フィード投稿（カルーセル、単一画像）
│   ├── ストーリーズコンテンツ
│   ├── リールのコンセプト
│   └── ビジュアル審美性のガイドライン
│
└── Reddit Community Builder → Reddit 戦略
    ├── サブレディットのターゲティング
    ├── 価値優先の投稿下書き
    ├── コメントエンゲージメント計画
    └── AMA の準備（該当する場合）
```

### 2週目: ローンチと起動

```
1日目: ローンチ前
├── すべてのコンテンツのキューと公開スケジュール設定完了
├── アナリティクストラッキングの確認完了
├── A/B テストバリアントの設定完了
├── ランディングページが稼働中かつテスト済み
└── エンゲージメントプロトコルについてチームへのブリーフィング完了

2〜3日目: ローンチ
├── Twitter Engager → ローンチスレッド + リアルタイムエンゲージメント
├── Instagram Curator → ローンチ投稿 + ストーリーズ
├── TikTok Strategist → ローンチ動画
├── Reddit Community Builder → 本物のコミュニティ投稿
├── Content Creator → ブログ投稿の公開 + メール一斉送信
├── Growth Hacker → 有料キャンペーンの起動
└── Analytics Reporter → リアルタイムダッシュボードモニタリング

4〜5日目: 最適化
├── Analytics Reporter → 最初の 48 時間パフォーマンスレポート
├── Growth Hacker → データに基づいたチャネル最適化
├── Experiment Tracker → A/B テストの早期結果
├── Social Media Strategist → エンゲージメント戦略の調整
└── Content Creator → 反響に基づいた対応コンテンツ
```

### 3〜4週目: 維持と最適化

```
日次:
├── プラットフォームエージェント → エンゲージメントとコンテンツ投稿
├── Analytics Reporter → 日次パフォーマンススナップショット
└── Growth Hacker → ファネル最適化

週次:
├── Social Media Strategist → キャンペーンパフォーマンスレビュー
├── Experiment Tracker → A/B テスト結果と新規テスト
├── Content Creator → パフォーマンスデータに基づいた新規コンテンツ
└── Analytics Reporter → 週次キャンペーンレポート

キャンペーン終了時:
├── Analytics Reporter → 包括的なキャンペーン分析
├── Growth Hacker → ROI 分析とチャネル効果
├── Executive Summary Generator → キャンペーンエグゼクティブサマリー
└── Social Media Strategist → 学びと推奨事項
```

## キャンペーンメトリクス

| メトリクス | 目標 | オーナー |
|--------|--------|-------|
| 総リーチ | [予算に基づく目標] | ソーシャルメディアストラテジスト |
| エンゲージメント率 | プラットフォーム平均 > 3% | プラットフォームエージェント |
| クリック率 | CTA で > 2% | グロースハッカー |
| コンバージョン率 | ランディングページ > 5% | グロースハッカー |
| 獲得単価 | < [目標 CAC] | グロースハッカー |
| ブランドセンチメント | ネットポジティブ | ブランドガーディアン |
| 公開コンテンツ数 | [目標数] | コンテンツクリエーター |
| 完了 A/B テスト数 | ≥ 5 | エクスペリメントトラッカー |

## プラットフォーム別 KPI

| プラットフォーム | 主要 KPI | 副次 KPI | エージェント |
|----------|------------|---------------|-------|
| Twitter/X | インプレッション + エンゲージメント率 | フォロワー増加 | Twitter エンゲージャー |
| TikTok | 視聴回数 + 完視率 | フォロワー増加 | TikTok ストラテジスト |
| Instagram | リーチ + 保存数 | プロフィール訪問 | Instagram キュレーター |
| Reddit | アップボート + コメント品質 | リファラルトラフィック | Reddit コミュニティビルダー |
| メール | 開封率 + CTR | 購読解除率 | コンテンツクリエーター |
| ブログ | オーガニックトラフィック + 滞在時間 | バックリンク | コンテンツクリエーター |
| 有料広告 | ROAS + CPA | 品質スコア | グロースハッカー |
